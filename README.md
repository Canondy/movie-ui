# Vue 3 + Vite

## 项目概要：使用Python的scrapy框架从豆瓣评分前250名的电影进行数据采集，存入到Mysql数据库，使用Java提供后端接口，前端使用Vue3

### 使用的编程语言：Python + Java + JavaScript

### 框架：Python(scrapy)  Java(SpringBoot)  JavaScript(Vue3)

### 这里只是电影项目的前端代码，手机端h5页面，使用Vite+Vue3脚手架构建
